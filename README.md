# AWS
Planejamento efetuado com base em uma loja que foi recentemente inaugurada.
Loja recem inaugurada que esta vendendo bastante mas, foi criada com computadores não eficazes para processamento local de dados.
Logo, tiveram a ideia de usar a nuvem para processar os dados, sendo assim que foi realizado.
Por questões de burocracia decidiram utilizar do S3, pois irão acessar bastante os dados enviados,
para poder verificar como estão as vendas e etc.. Os arquivos sao enviados do computador fisico da loja de forma compactada,
descompactado nas VMs e enviado para processamento e armazenamento na nuvem.
