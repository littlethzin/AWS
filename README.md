# AWS
Planejamento efetuado com base em uma loja que foi recentemente inaugurada, que esta vendendo bastante produtos.
Loja recem aberta e foi criada com computadores nao tao bons para processamento local de dados,
logo, tiveram a ideia de usar a nuvem para processar os dados, sendo assim que foi realizado.
Por questões de burocracia decidiram utilizar do S3, pois irão acessar bastante os dados enviados,
para poder verificar como estão as vendas e etc.. Os arquivos sao enviados de forma compactada
descompactado nas VMs e enviado para processamento e armazenamento na nuvem.
